Let's imitate native iOS and Android pull-to-refresh behavior. This component should listen to user's touch events and gradually shows loader from the top (like in Android or iOS). When user has pulled it far enough, it should emit a (pulled) event to start loading and continue showing loaders until LOADED event happens. You need to write code in pull-to-refresh component to implement this logic. Feel free to take a look at real Android / iOS devices to imitate their behavior closer.
